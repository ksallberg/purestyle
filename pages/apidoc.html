Play API docs

Register (POST)

curl example:

curl -d "username=USER&password=PASS" -c - https://play.purestyle.se/api/register

response example (Note, session cookie called username is set):

{"status":"success"}

response example (fail):

{"status":"error","error":"User already exists"}

Login (POST)

curl example:

curl -d "username=USER&password=PASS" -c - https://play.purestyle.se/api/login

response example (Note, session cookie called username is set):

{"status":"success"}

response example (fail):

{"status":"error"}

Logout (GET)

curl example (Note, session cookie is cleared):

curl --cookie "username=$cookie_username" -c - https://play.purestyle.se/api/logout

response example:

{"status":"success"}

Get playlists (GET)

curl example:

curl --cookie "username=$cookie_username" https://play.purestyle.se/api/playlists

response example:

{"lists":[{"id":"some-id","name":"playlist1"}]}

Create playlist (POST)

curl example:

curl --cookie "username=$cookie_username" -d "playlist_name=playlist1" https://play.purestyle.se/api/playlists

response example:

{"id":"some-id","name":"playlist1","status":"success"}

response example (nog logged in):

{"status":"error","message":"Access denied"}

Get tracks of playlist (GET)

curl example:

curl --cookie "username=$cookie_username" https://play.purestyle.se/api/playlist?list=PLAYLISTID

response example:

{"id":PLAYLISTID,"name":"playlist1","tracks":[{"id":TRACKID,"source":"other","title":"apa","url":"apa"}]}

Add track to playlist (POST)

curl example:

curl --cookie "username=$cookie_username" -d "playlist=PLAYLISTID" -d "track_name=apa" https://play.purestyle.se/api/playlist?list=PLAYLISTID

response example:

{"status"\:"success","track"\:{"id"\:(.*),"source"\:"other","title"\:"apa","url"\:"apa"}

response example (nog logged in):

{"status":"error","message":"Access denied"}
